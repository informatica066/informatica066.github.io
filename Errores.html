Recall error con windows 11

Recall

  • Hace capturas de pantalla en nuestro ordenador para poder mejorar la experiencia con la inteligencia artificial.
  • Lo recomendable seria quitarlo
  • Para poder quitarlo hay que hacer lo siguiente
  • Mirar si lo tenemos quitado ya que la version donde esta implementado es en la version de windows 24h2
  • Damos al boton windows escribimos cmd y lo ejecutamos como administrador
  • Una vez dentro comprobamos si esta activado para ello hacemos lo siguiente
  • dism /online /Get-featureinfo /Featurename:Recall con este comando comprabamos si esta activo que nos saldra Recall:Enable
  • Si es asi para desactivarlo escribimos lo siguiente dism /online /Disable-feature /Featurename:recall
  • Nos pidira que reiniciemos una vez reiniciamos ya lo tendremos desactivado
